# AI拍照亭 — 产品需求文档 v1.0

## 一、产品概述

**产品定位：** 便携桌面AI拍照亭。用户拍照或上传照片后，AI自动将照片转换为卡通/贴纸风格，选择尺寸后一键打印成贴纸、钥匙扣等周边产品。形态类似迷你 Photo Booth，目标成为"照片变周边的最后1米"。

**核心价值：** 把"照片变周边"从"打开PS、找素材、抠图调色、排版打印"的繁琐流程，压缩为"拍一张 → 按一下 → 拿到贴纸"的90秒体验。

**产品Slogan：** 拍一张，就变贴纸。

**目标用户：**

| 用户群 | 场景 | 核心诉求 |
|--------|------|---------|
| 喜欢做周边的年轻人 | DIY手账贴纸、手机壳装饰 | 快速、个性化、好玩 |
| 漫展/同人展摊主 | coser现场拍照做钥匙扣 | 即时交付、风格多样 |
| 奶茶店/文创店 | 顾客消费后打印纪念贴纸 | 引流、增加客单价 |
| 朋友聚会/情侣 | 合影→卡通头像→贴纸留念 | 社交分享、仪式感 |

**市场定位：** 现有竞品（如拍立得、淘宝定制贴纸）的问题是：拍立得没有AI风格转换，淘宝定制要等好几天。AI拍照亭填补"即时AI定制 + 即时拿到实物"的空白。

## 二、用户场景

### 场景1：漫展现场

> "小A是coser，穿着精心准备的明日香cos服逛漫展。看到我们摊位，走过来，站在拍照亭前，选'日系动漫'风格，3秒倒计时拍了一张。AI生成3版让她挑，她选了最喜欢的那版，点打印，15秒后一张动漫风格钥匙扣从机器里吐出来。她惊呼'这就是我！'然后发朋友圈。"

**关键触点：** 拍照体验要快（≤10秒）、AI结果要像本人、打印要即时

### 场景2：闺蜜聚会

> "小B和小C周末约下午茶，用拍照亭选了'美式漫画'风格，两人自拍合照。AI生成了一组漫画分镜风格的图，她们分别打印了两张贴纸，贴在各自的手机壳上。"

**关键触点：** 支持双人/多人自拍、生成效果有趣味性、可多份打印

### 场景3：宠物咖啡馆

> "一家猫咖老板在店里放了台拍照亭。顾客撸猫时拍了张猫的照片，选'Q版萌宠'风格，几秒后AI把猫画成了超可爱的卡通形象，顾客扫码付3块钱打印贴纸带走。"

**关键触点：** 宠物识别效果好、支持付费、设备稳定无人值守

### 场景4：文创市集

> "周末文创市集，摊位主摆了一台拍照亭，旁边的牌子写着'AI定制你的专属贴纸 ¥5/张'。路过的人好奇围观，有人试了试，拿到贴纸后很开心，又拉朋友来拍。"

**关键触点：** 吸引眼球的设备外观、简单的付费流程、围观效应

## 三、功能需求

### P0 — 核心闭环（MVP必须实现）

**1. 照片输入**
- 调用设备摄像头实时预览，支持切换前后摄像头
- 倒计时3秒拍照（防抖）+ 快拍按钮
- 从相册/文件管理器选择已有照片
- 照片本地压缩后再上传（≤1024px，JPEG85%质量，控制文件大小）

**2. AI风格转换**
- 支持6种预设风格：日系动漫、美式漫画、像素复古、Q版萌宠、水彩手绘、治愈鼓励
- 用户可选填自定义风格描述
- 后台两步处理：Qwen-VL分析照片特征 → 万相图生图做风格迁移
- 当前使用阿里通义万相API；预留其他AI引擎接口（OpenAI DALL-E、Stable Diffusion等）

**3. 多次生成与选择**
- 每次上传照片后最多生成3次
- 3次结果全部缓存（前端内存 + 后端文件），用户可切换缩略图查看
- 选中一张最满意的进入打印流程
- 第3次生成后"重新生成"按钮禁用

**4. 预览与决策**
- 大图预览，下方缩略图切换条
- "选择此张打印" → 进入尺寸选择
- "重新生成" → 再来一次（显示剩余次数）
- "不打印了" → 确认后返回首页

**5. 打印输出**
- 4种尺寸：钥匙扣(30mm圆)、小贴纸(57×57mm)、大贴纸(76×76mm)、明信片(100×148mm)
- 可选打印份数（默认1份）
- 后台通过WebSocket发送到PC打印中继 → Windows系统调用打印机

### P1 — 体验打磨

**6. 打印进度追踪**
- 实时展示状态：排队中→发送到打印机→打印机处理中→打印完成/失败
- 进度条 + 文字状态 + 预估等待时间
- 失败时显示原因（缺纸、离线、卡纸）和重试按钮
- 支持中途取消

**7. 生成进度动画**
- 显示AI处理进度："分析照片特征… → 正在风格迁移… → 优化细节…"
- 每步完成打勾，当前步有loading动效

**8. 相册历史**
- 查看所有历史生成记录（网格/瀑布流）
- 每条记录显示：缩略图、风格名、日期
- 点击可放大查看、再次打印

**9. 风格管理**
- 增删改风格配方
- 预设6种风格不可删除但可编辑
- 每种风格包含：名称、图标、分类、提示词模板、版本号
- 生成后可打分（1-5星），高分配方自动标记为成熟

**10. 分享功能**
- 生成分享链接/二维码
- 朋友扫码可查看生成的图片
- 分享内容带水印+来源

### P2 — 社交与商业化

**11. 双人模式**
- 两个朋友同时站在镜头前，AI识别双人并生成合照卡通图

**12. 付费打印**
- 微信支付/支付宝扫码付费
- 管理员可设置不同尺寸的价格

**13. 数据看板**
- 打印总量、热门风格排行、设备在线状态

### P3 — 硬件整合（未来）

- 蓝牙热敏打印机直连（去掉PC依赖）
- 3D打印外壳（饭盒大小，手机+打印一体）
- 外接物理拍照按钮（蓝牙遥控器方案）

## 四、页面结构

### 页面1：首页 /

- 顶部：Logo + Slogan"AI拍照亭"
- 中部：风格选择区域，2列网格展示6种预设风格卡片。每个卡片包含emoji图标、名称、简短描述。选中态高亮（紫色边框+浅紫底色）
- 底部：「自定义风格描述」输入框（选填）→ 「开始拍照」（主按钮，大圆角填充）→ 「从相册选择」（次按钮，描边）
- 最近生成入口（若有历史记录则显示最近3张缩略图）

### 页面2：拍照页 /capture

- 全屏摄像头画面（前置/后置可切换）
- 顶部栏：返回按钮 + 当前风格名（半透明胶囊）+ 切换摄像头按钮
- 底部操作区：
  - 左侧：相册入口（小图标按钮）
  - 中间：快门大按钮（白色外圈+实心圆，点击倒计时3-2-1拍照）
  - 右侧：快拍按钮（小按钮，跳过倒计时立即拍）
- 拍照瞬间：屏幕闪白一次

### 页面3：生成中（内嵌状态）

- 覆盖在预览页上
- 动画转圈 + 分步进度文字
- "第 1/3 次生成"标识
- 取消按钮（× 右上角）

### 页面4：预览页 /preview

- 主图区（正方形，居中，圆角，大图显示当前选中的生成结果）
- 缩略图切换条（横向排列3个圆形缩略图，已生成的亮色，未生成的灰色占位，当前选中放大+紫色边框）
- 重新生成按钮（显示"还剩2次"/"还剩1次"，0次变灰）
- 选择此张打印（主按钮，渐变紫色）
- 不打印了（底部文字链接，灰色）

### 页面5：尺寸选择（弹窗）

- 半屏弹窗从底部滑出
- 4种尺寸卡片（钥匙扣/小贴纸/大贴纸/明信片），每卡片显示emoji+名称+尺寸规格
- 份数选择器（- 1 +）
- 底部：取消 + 确认打印 双按钮

### 页面6：打印中

- 居中打印机动画（SVG，模拟出纸）
- 进度条（百分比+长度）
- 状态文字："正在打印…请稍候"
- 打印详情：尺寸、份数、打印机名称
- 取消打印按钮

### 页面7：打印结果

- 成功：绿色大勾 + "打印完成！" + 纸张飞出动画 + "再来一张"按钮
- 失败：红色× + 错误信息 + "重试"按钮 + "返回首页"

### 页面8：相册 /gallery

- 2列瀑布流网格
- 每张卡片：生成图缩略图 + 风格标签(小圆角) + 日期
- 点击：全屏查看、再次打印
- 空状态：插画+文案"还没有生成过，去拍一张吧~"

### 页面9：风格管理 /styles

- 列表展示所有风格
- 每项：emoji + 名称 + 分类标签 + 使用次数 + 评分星星 + 编辑/删除按钮
- 顶部「新建风格」按钮
- 编辑弹窗：emoji、名称、分类下拉、提示词模板文本框

### 页面10：管理员 /admin（P2）

- 密码保护
- 纸张尺寸管理（增删改，宽mm、高mm、形状）
- 打印历史（表格：时间、缩略图、尺寸、状态）
- 中继连接状态（在线/离线，最后心跳时间）

### 视觉系统

| 属性 | 值 |
|------|-----|

| 圆角 | 小元素12px、卡片16px、按钮20px、弹窗24px |
| 阴影 | 主按钮 shadow-lg shadow-primary/30 |

### 按钮规范

- 主操作：大圆角填充按钮（w-full py-4，bg-primary，白色文字，active:scale-95）
- 次操作：描边按钮（border border-white/5，bg-surface-card）
- 危险操作：文字按钮（text-text-muted，hover变红）
- 禁用态：opacity-50

### 反馈规范

- 所有异步操作有loading态（旋转圈或骨架屏）
- 成功反馈：绿色提示条或成功图标动画
- 失败反馈：红色提示 + 错误说明 + 恢复路径（重试/返回）
- 空状态：插画emoji + 引导文案 + 行动按钮
- 确认操作：弹窗二次确认（如"放弃所有生成结果？"）

### 手势（P1+）

- 左右滑动切换风格卡片
- 上滑/下滑切换照片（预览页）
- 长按快门按钮 → 预留录像功能

### 边界处理

| 边界情况 | 处理方式 |
|---------|---------|
| 摄像头权限被拒 | 全屏提示"请在设置中允许相机权限" + 使用相册按钮 |
| HTTPS不可用（开发环境） | 提示使用localhost访问或部署到Vercel |
| AI生成超时(>2分钟) | "生成时间较长，请稍候" + 取消按钮 + 自动重试1次 |
| 生成结果不满意 | 最多3次重新生成，缓存切换 |
| 没有打印机连接 | "PC打印服务未连接，任务已排队" |
| 打印机缺纸/离线 | "打印失败：打印机缺纸，请装纸后重试" |
| 网络断开 | 错误提示 + 重试按钮（离线时相册仍可浏览） |
| 大图片上传 | 前端压缩至≤1024px后再上传，限制最大15MB |

|----|------|------|------|-----------|

| chibi | Q版萌宠 | 🐾 | pet | Q版卡哇伊风格，圆润造型，大头小身体，闪闪大眼睛，粉彩色调 |
| watercolor | 水彩手绘 | 🎨 | portrait | 水彩手绘风格，柔和色彩晕染，毛笔笔触，艺术感 |

### 风格配方迭代流程

创建初版提示词 → 测试生成 → 打分(1-5星) → 1-2星废弃或大改，3星标记可用微调，4-5星锁定为成熟配方

### 评分自动优化

系统记录每种风格的平均分、评分次数、使用总次数、最近10次生成评分趋势。高分配方自动排前，低分配方建议优化或废弃。

## 七、打印系统设计

### Phase 1 架构（当前）

前端(PWA) → POST /api/print → 后端(WebSocket) → PC打印中继(Python) → Windows系统打印API → 佳能/喷墨打印机

### 打印状态机

PENDING → SENDING → PRINTING → DONE / ERROR(可重试) / CANCELLED

### 打印尺寸体系

| ID | 名称 | 宽mm | 高mm | 形状 | 排版说明 |
|----|------|------|------|------|---------|
| keychain | 钥匙扣 | 30 | 30 | circle | 圆形裁切，300DPI输出 |
| sticker_s | 小贴纸 | 57 | 57 | rect | 圆角矩形+切割线 |
| sticker_l | 大贴纸 | 76 | 76 | rect | 圆角矩形+切割线 |
| card | 明信片 | 100 | 148 | rect | 全幅无边距 |

### PC打印中继特性

Python脚本，Windows托盘常驻。启动时自动连接后端WebSocket，断线自动重连（5秒间隔）。接收任务 → 下载图片 → 按尺寸排版 → 调系统打印。托盘菜单：查看打印机列表、退出。多台PC可同时连接，任务发给最先连上的。

### 错误处理

| 错误 | 原因 | 处理 |
|------|------|------|
| 缺纸 | 打印机纸张用尽 | 提示用户装纸，任务保留可重试 |
| 离线 | 打印机关机或断连 | 标记失败，提示检查打印机 |
| 卡纸 | 纸张卡住 | 提示用户清理卡纸后重试 |
| 中继离线 | PC未运行打印服务 | 任务排队，等中继上线自动发送 |

## 九、非功能需求

| 类别 | 要求 |
|------|------|
| **性能** | AI生成响应≤60秒；打印指令发送≤3秒；页面加载≤2秒 |
| **兼容性** | iOS Safari 15+、Android Chrome 100+、桌面 Chrome/Edge 100+；PWA可安装 |
| **安全** | API Key仅存服务端环境变量；管理员页面密码保护；图片上传限制15MB |
| **可用性** | 首次使用引导；所有关键操作有确认/撤销；错误有明确恢复路径 |
| **可维护性** | 前后端分离；AI引擎可插拔（切换模型只改环境变量）；日志记录关键操作 |
| **成本控制** | 优先用免费额度（通义万相每日50张）；图片存储自动清理30天前文件 |

## 十二、风险与应对

| 风险 | 影响 | 概率 | 应对策略 |
|------|------|------|---------|
| AI生成效果不稳定，不像本人 | 用户流失 | 高 | 最多3次生成+风格评分系统+后续换更先进的模型 |
| 打印机兼容性差 | 打印失败 | 中 | 先适配佳能和主流喷墨再扩展；PC中继抽象打印接口 |
| 通义API费用上涨或停服 | 成本增加 | 低 | 预留OpenAI/Stable Diffusion接口，一键切换 |
| 手机摄像头权限获取困难 | 无法拍照 | 中 | 引导文案+相册fallback（选照片也能用） |
| 热敏打印效果不如照片打印机 | 品质不满 | 中 | Phase 1用照片打印机保证品质；热敏仅做便携方案 |
| 设备被恶意使用生成不当内容 | 合规风险 | 低 | 通义自带内容审核；前后端关键词过滤；打印记录可追溯 |
