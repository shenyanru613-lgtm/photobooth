## 二、详细流程

### ① 选择照片

**入口：** 首页 → 「开始拍照」或「从相册选择」

**拍照模式：**
- 用户看到全屏摄像头画面
- 切换前/后摄像头
- 拍照方式：倒计时3秒 / 快拍（立即拍）
- 拍照后自动进入②

**相册模式：**
- 弹出系统相册选择
- 选好后自动进入②

**状态：**
```
照片来源: camera | album
原始照片: { file, dataUrl }  // 存在前端内存
```

---

### ② AI 生成（最多 3 次）

**流程：**

```
第1次生成
  ├── 上传照片 + 风格ID → 后端
  ├── 显示生成进度动画（识图中... → 绘制中...）
  ├── 结果返回 → 进入③预览
  │
  ├── 用户点「重新生成」→ 第2次
  │    ├── 同样流程
  │    └── 结果返回 → 进入③预览
  │
  ├── 用户点「重新生成」→ 第3次（最后一次）
  │    ├── 同样流程
  │    └── 结果返回 → 进入③预览
  │
  └── 3次用完，「重新生成」按钮变灰/隐藏
```

**缓存策略：**
```
generationResults = [
  { id: 1, imageUrl: '...', prompt: '...', createdAt: '...' },
  { id: 2, imageUrl: '...', prompt: '...', createdAt: '...' },
  { id: 3, imageUrl: '...', prompt: '...', createdAt: '...' },
]
// 存在前端内存，切换查看不重新请求
// 后端也保留所有3张的图片文件
```

**生成中状态：**
```
┌─────────────────────────────────┐
│         🔄 AI 正在生成...        │
│                                 │
│    ◉ 正在分析照片特征...   ✅    │
│    ◉ 正在应用风格转换...   ⏳    │
│    ○ 正在优化细节...             │
│                                 │
│    第 1/3 次生成                │
│    [取消]                       │
└─────────────────────────────────┘
```

---

### ③ 预览 & 决策（核心交互）

**页面布局：**

```
┌─────────────────────────────────┐
│  ← 返回    预览结果    第1/3次    │
│                                 │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │     AI 生成的大图        │    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  风格: 日系动漫                   │
│                                 │
│  ┌─结果选择器──────────────────┐  │
│  │ [缩略图1] [缩略图2] [缩略图3] │  │
│  │   当前     等待     未生成    │  │
│  └────────────────────────────┘  │
│                                 │
│  ┌──────────┐ ┌──────────────┐  │
│  │ 重新生成  │ │  选择此张打印  │  │
│  │ (2/3剩余) │ │     →        │  │
│  └──────────┘ └──────────────┘  │
│                                 │
│  [不打印了，返回首页]              │
└─────────────────────────────────┘
```

**交互规则：**

| 用户操作 | 行为 |
|---------|------|
| 点击缩略图切换 | 主图切换为该次结果（已生成的才能点） |
| 点击「重新生成」 | 当前次数+1 → 回到②，新结果追加到缓存 |
| 点击「选择此张打印」 | 锁定当前选中图 → 进入④选尺寸 |
| 点击「不打印了」 | 清空缓存 → 回到首页 |
| 返回按钮 | 弹窗确认「放弃所有生成结果？」→ 是 → 回到首页 |

**边界条件：**
- 第1次生成结果不满意 → 点「重新生成」→ 第2次
- 第3次生成后 → 「重新生成」按钮禁用，显示「已达最大次数」
- 任何时候都能点「选择此张打印」（用当前显示的那张）
- 任何时候都能点「不打印了」

### ⑤ 打印 & 进度

**打印流程：**

```
确认打印
  │
  ├── 前端 → 后端 POST /api/print { generationId, size, copies }
  │
  ├── 后端 → WebSocket 发给 PC 打印中继
  │     │
  │     ├── 中继连接正常：
  │     │   ├── 下载图片
  │     │   ├── 按尺寸排版
  │     │   ├── 发送到打印机
  │     │   └── 返回结果（成功/失败）
  │     │
  │     └── 中继未连接：
  │         └── 任务排队，等中继上线后自动打印
  │
  └── 前端实时显示进度
```

**打印进度界面：**

```
┌─────────────────────────────────┐
│         🖨️ 正在打印...          │
│                                 │
│    ┌─────────────────────┐      │
│    │ ████████████░░░░░░░ │ 60%  │
│    └─────────────────────┘      │
│                                 │
│    ✅ 已发送到打印机              │
│    ⏳ 打印机正在处理...           │
│                                 │
│    打印机: Canon SELPHY          │
│    尺寸: 钥匙扣 (30mm圆)         │
│    份数: 1                      │
│                                 │
│    [取消打印]                    │
└─────────────────────────────────┘
```

**打印结果状态：**

| 状态 | 显示 |
|------|------|
| `pending` | 打印任务已排队，等待 PC 连接... |
| `sending` | 正在发送到打印机... |
| `printing` | 打印机正在出纸... |
| `done` | ✅ 打印完成！贴纸已出。 |
| `error` | ❌ 打印失败：缺纸 / 卡纸 / 打印机离线 |
| `cancelled` | 已取消打印 |

**状态来源：**
```
后端 WebSocket 事件:
  - print_status { jobId, status: 'pending'|'sending'|'printing'|'done'|'error', message }
  - 中继 emit: print_done / print_error
  - 后端 broadcast: print_status
```

                       │ 生成完成                 │ 重新生成(count<3)

              ┌──────────────────┐               │

         ┌──────────┐  ┌─────────┐

              │ 确认打印
              ↓
         ┌──────────┐
         │ PRINTING │ 打印中 + 进度

              │
         ┌────┴────┐
         ↓         ↓

         ↓         ↓
    ┌────────┐ ┌──────────────┐

## 五、前后端数据流

```
前端                         后端                       PC中继
 │                            │                          │
 │── POST /api/generate ─────→│                          │
 │   (photo + styleId)        │                          │
 │                            │── 通义API ──→            │
 │←── { generationId,        │                          │
 │      imageUrl } ──────────│                          │
 │                            │                          │
 │  (可重复3次，每次不同       │                          │
 │   generationId)            │                          │
 │                            │                          │
 │── POST /api/print ────────→│                          │
 │   { generationId,          │                          │
 │     size, copies }         │── WebSocket ────────────→│
 │                            │   { jobId, imageUrl,     │
 │                            │     size, copies }       │
 │                            │                          │── 下载图片
 │                            │                          │── 排版
 │                            │                          │── 打印
 │                            │←── print_done ──────────│
 │←── WebSocket ──────────────│                          │
 │   print_status: done       │                          │
```
